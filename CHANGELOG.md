# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.5] - 2026-03-22 (MVP v2.5)

### Added
- Comprehensive repository documentation in `docs/` folder.
- Project logo and enhanced README.
- Quality attribute scenarios for performance and reliability.
- User Acceptance Testing (UAT) results documented.
- Markdown templates for issues and pull requests.

### Fixed
- Voice recitation accuracy scoring for Russian poems.
- Minor navigation bugs in the Telegram bot interface.

## [0.2.0] - 2026-03-15 (MVP v2)

### Added
- Speech-to-Text (Vosk) integration for voice check.
- Spaced Repetition (SM-2) algorithm for memorization scheduling.
- PostgreSQL database integration for persistent storage.
- Docker & Docker Compose setup for easy deployment.

### Changed
- Refactored API to use `/api/v1` prefix.
- Improved poem recommendation logic.

## [0.1.0] - 2026-03-01 (MVP v1)

### Added
- Initial Telegram bot interface via `aiogram`.
- Basic poem recommendation (static list).
- Project goal definition and initial requirements gathering.

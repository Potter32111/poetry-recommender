# AI Usage Report (Assignments 1-7)

This report documents the use of Large Language Models (LLMs) and AI agents throughout the development of the "Poetry Recommender" project.

## Assignment 1: Deciding what to build
- **Usage:** Used Gemini to brainstorm poetry-related project ideas and refine the initial interview script. AI helped apply "The Mom Test" principles by identifying potentially biased questions.

## Assignment 2: Plan, design, deploy
- **Usage:** AI assisted in drafting 15+ user stories based on interview notes. It also generated the initial PlantUML code for the Entity-Relationship Diagram (ERD).

## Assignment 3: Sprint, Git Workflow, MVP v1
- **Usage:** Used an AI agent to structure the GitHub Milestone and suggest a clean branching strategy (GitHub Flow). AI also helped draft the initial Definition of Done (DoD).

## Assignment 4: Quality Assurance & Automation
- **Usage:** AI was instrumental in writing `pytest` test cases for asynchronous FastAPI endpoints. It also helped configure the GitHub Actions pipeline for linting and testing.

## Assignment 5: Quality Attributes and Architecture
- **Usage:** LLMs were used to define Quality Attribute Scenarios (ISO 25010) and to refine the UML Component and Sequence diagrams. AI also generated issue and PR templates.

## Assignment 6: Making your repo great
- **Usage:** AI helped polish the README.md content, ensuring a professional tone. It also assisted in generating the CHANGELOG.md based on commit messages.

## Assignment 7: Delivering a valuable product
- **Usage:** AI agent (Antigravity) was used to synthesize the final transition report, summarize meeting notes, and generate the final LaTeX report for submission.

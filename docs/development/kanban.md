# Kanban Board

## Access
- **Link**: [GitHub Project Board](https://github.com/Potter32111/poetry-recommender/projects)

## Column Entry Criteria

### 📥 To Do
- Issue is clearly defined.
- Labels are applied (e.g., `feature`, `bug`, `debt`).
- Priority is set.

### ⚙️ In Progress
- Issue is assigned to a team member.
- A dedicated branch exists for the task.
- Target milestone is set.

### 🔍 In Review
- Pull Request is created and linked to the issue.
- All CI tests are passing.
- Code is ready for peer review.

### 🧪 User Testing
- Feature is deployed to the staging/test environment.
- Acceptance criteria are verified.
- Feedback is documented.

### ✅ Done
- Code is merged into `main`.
- Issue is closed.
- Documentation is updated.
